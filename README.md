# AUDIO AR GAME MAKER
音が鳴るARオブジェクトを実空間に配置することで、一人あるいは他の人と一緒に遊びを作れるツール。

## Requirement
- Unity 2021.3.16f1
- PUN 2 (Photon Unity Networking 2) Version 2.41
- Resonance Audio SDK for Unity v1.2.1
